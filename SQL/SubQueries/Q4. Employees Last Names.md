# Q4. Employees Last Names


## Problem Statement:

Write a query to return a list of all employees with selected last names: King, Taylor, and Grant from the employee's table.

Return all the fields.
Return the result ordered by employee_id in ascending order.
Sample Input:

Table: employees
![image](https://github.com/vishwasjoshi2019/DSML/assets/98074283/a23adfb1-37b0-46a7-b918-747e1e691af1)

Sample output:
![image](https://github.com/vishwasjoshi2019/DSML/assets/98074283/991bf75a-0502-4f28-9d64-44d56e7482e9)



---

## Solution

```sql
SELECT * FROM
employees
where last_name in("King","Taylor","Grant")
order by employee_id
```
