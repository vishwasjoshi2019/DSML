# Date from string

## How would you get the date from the below string?

'2019-08-06 08:10:12'

SELECT ExtractDATE('2019-08-06 08:10:12' )
SELECT GetDate('2019-08-06 08:10:12')
SELECT TakeDate('2019-08-06 08:10:12')
SELECT Date('2019-08-06 08:10:12')
