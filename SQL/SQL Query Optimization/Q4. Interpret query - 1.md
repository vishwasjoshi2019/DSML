# Interpret query - 1


## What is the interpretation of the given SQL statement?
``` sql
SELECT DATE_ADD("2017-06-15", INTERVAL 10 DAY);
```
- Add 10 days to the given date
- Add 10 hours to the given date
- Add 1 year to the current date
- None of the abov
---

## Solution

```
Add 10 days to the given date
```
