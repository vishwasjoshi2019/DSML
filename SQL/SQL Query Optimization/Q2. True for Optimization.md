# True for Optimization

## Which of the statements is true about query optimization?
- Query optimization is only necessary for large databases
- Query optimization is not important for modern databases
- Query optimization is a critical part of the database design process
- Query optimization should be performed only by database administrators
---

## Solution

```
Query optimization is a critical part of the database design process
```
